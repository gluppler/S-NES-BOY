# 4.2 PPU Cheatsheets

## System Applicability

**This cheatsheet applies to:**
- ✅ **Game Boy (DMG): Primary focus
- ✅ **Game Boy Color (CGB): Differences explicitly labeled

## LCD Registers

| Address | Name | Write | Read | Bits |
|---------|------|-------|------|------|
| $FF40 | rLCDC | LCD Control | — | LCD P W B S O B G |
| $FF41 | rSTAT | LCD Status | Status | L - - M M M C |
| $FF42 | rSCY | Scroll Y | — | Y Y Y Y Y Y Y Y |
| $FF43 | rSCX | Scroll X | — | X X X X X X X X |
| $FF44 | rLY | LY | Current Line | L L L L L L L L |
| $FF45 | rLYC | LYC | — | L L L L L L L L |
| $FF46 | rDMA | DMA | — | A A A A A A A A |
| $FF47 | rBGP | BG Palette | — | P P P P P P P P |
| $FF48 | rOBP0 | OBJ Palette 0 | — | P P P P P P P P |
| $FF49 | rOBP1 | OBJ Palette 1 | — | P P P P P P P P |
| $FF4A | rWY | Window Y | — | Y Y Y Y Y Y Y Y |
| $FF4B | rWX | Window X | — | X X X X X X X X |

## LCD Modes

| Mode | Name | Duration | VRAM | OAM |
|------|------|----------|------|-----|
| 0 | HBlank | 87-204 cycles | ✅ | ✅ |
| 1 | VBlank | 4560 cycles | ✅ | ✅ |
| 2 | OAM Search | 80 cycles | ✅ | ❌ |
| 3 | Pixel Transfer | 172-289 cycles | ❌ | ❌ |

## LCD Control (rLCDC) Bits

| Bit | Name | Meaning |
|-----|------|---------|
| 7 | LCD | LCD enable (1 = on) |
| 6 | W | Window tilemap (1 = $9C00) |
| 5 | W | Window enable (1 = on) |
| 4 | B | BG/Window tile data (1 = $8000) |
| 3 | B | BG tilemap (1 = $9C00) |
| 2 | S | Sprite size (1 = 8×16) |
| 1 | O | Sprite enable (1 = on) |
| 0 | G | BG enable (1 = on) |

## LCD Status (rSTAT) Bits

| Bit | Name | Meaning |
|-----|------|---------|
| 7 | L | LYC=LY interrupt |
| 6 | L | LYC=LY flag |
| 5-3 | — | Unused |
| 2 | M | Mode 2 interrupt |
| 1-0 | M | Mode (0-3) |

## Palettes (DMG)

| Register | Colors | Format |
|----------|--------|--------|
| rBGP | BG 4-shade | %11100100 = White, Light Gray, Dark Gray, Black |
| rOBP0 | Sprite 0 | %11100100 = White, Light Gray, Dark Gray, Black |
| rOBP1 | Sprite 1 | %11100100 = White, Light Gray, Dark Gray, Black |

## Palettes (CGB)

| Register | Description |
|----------|-------------|
| rBCPS | BG palette index/auto-increment |
| rBCPD | BG palette data (15-bit RGB) |
| rOCPS | OBJ palette index/auto-increment |
| rOCPD | OBJ palette data (15-bit RGB) |

## VRAM Layout

| Address | Size | Description |
|---------|------|-------------|
| $8000-$97FF | 6 KB | Tile data (384 tiles) |
| $9800-$9BFF | 1 KB | BG tilemap (32×32) |
| $9C00-$9FFF | 1 KB | Window tilemap (32×32) |

## OAM Format

| Byte | Offset | Bits | Meaning |
|------|--------|------|---------|
| 0 | Y | 0-7 | Y position (0-159) |
| 1 | X | 0-7 | X position (0-167) |
| 2 | Tile | 0-7 | Tile number (0-255) |
| 3 | Attr | 7 | Priority (0 = front) |
| 3 | Attr | 6 | Y flip (1 = flip) |
| 3 | Attr | 5 | X flip (1 = flip) |
| 3 | Attr | 4 | Palette (CGB: 0-7) |
| 3 | Attr | 3-0 | Palette (DMG: 0-1) |

## Sprite Limits

| Limit | Value |
|-------|-------|
| Total sprites | 40 |
| Sprites per scanline | 10 |
| Sprite size (normal) | 8×8 |
| Sprite size (large) | 8×16 |
