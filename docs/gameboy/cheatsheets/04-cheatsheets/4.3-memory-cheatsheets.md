# 4.3 Memory Cheatsheets

## System Applicability

**This cheatsheet applies to:**
- ✅ **Game Boy (DMG): Primary focus
- ✅ **Game Boy Color (CGB): Differences explicitly labeled

## Memory Map

| Address Range | Size | Description | Access |
|--------------|------|-------------|--------|
| $0000-$3FFF | 16 KB | ROM Bank 0 | Always |
| $4000-$7FFF | 16 KB | ROM Bank N | Always (banked) |
| $8000-$9FFF | 8 KB | VRAM | Mode 0/1 only |
| $A000-$BFFF | 8 KB | External RAM | MBC-controlled |
| $C000-$CFFF | 4 KB | WRAM Bank 0 | Always |
| $D000-$DFFF | 4 KB | WRAM Bank 1 | Always (CGB banked) |
| $E000-$FDFF | 8 KB | WRAM Mirror | Always |
| $FE00-$FE9F | 160 B | OAM | Mode 0/1 only |
| $FEA0-$FEFF | 96 B | Unusable | Do not access |
| $FF00-$FF7F | 128 B | I/O Registers | Timing-dependent |
| $FF80-$FFFE | 127 B | HRAM | Always (fastest) |
| $FFFF | 1 B | IE Register | Always |

## Memory Access Timing

| Memory | Read Cycles | Write Cycles | Notes |
|--------|-------------|--------------|-------|
| HRAM | 1 | 1 | Fastest |
| WRAM | 2 | 2 | Standard |
| VRAM | 2 | 2 | Mode 0/1 only |
| OAM | 2 | 2 | Mode 0/1 only |
| ROM | 2 | N/A | Read-only |

## VRAM Access Restrictions

| LCD Mode | VRAM Access | OAM Access |
|----------|-------------|------------|
| 0 (HBlank) | ✅ | ✅ |
| 1 (VBlank) | ✅ | ✅ |
| 2 (OAM Search) | ✅ | ❌ |
| 3 (Pixel Transfer) | ❌ | ❌ |

## CGB Memory Banking

| Type | Banks | Switch Register | Notes |
|------|-------|------------------|-------|
| VRAM | 2 | rVBK ($FF4F) | 16 KB total |
| WRAM | 8 | rSVBK ($FF70) | 32 KB total |

## MBC Types

| MBC | ROM Banks | RAM Banks | Features |
|-----|-----------|-----------|----------|
| None | 2 | 0 | ROM only |
| MBC1 | 128 | 4 | Basic banking |
| MBC2 | 16 | 1 | Built-in RAM |
| MBC3 | 128 | 4 | RTC support |
| MBC5 | 512 | 16 | Large ROM/RAM |
