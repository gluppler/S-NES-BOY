# 4.4 Timing Cheatsheets

## System Applicability

**This cheatsheet applies to:**
- ✅ **Game Boy (DMG): Primary focus
- ✅ **Game Boy Color (CGB): Differences explicitly labeled

## Frame Timing

| Period | Duration | Cycles | Description |
|--------|----------|--------|-------------|
| Frame | 16.67 ms | 59,727 | Complete frame |
| VBlank | 1.09 ms | 4,560 | Vertical blanking |
| Visible | 15.58 ms | 55,167 | Visible rendering |
| Scanline | 114 μs | 456 | Per scanline |

## LCD Mode Timing

| Mode | Duration | Cycles | Description |
|------|----------|--------|-------------|
| 0 (HBlank) | 19-45 μs | 87-204 | Horizontal blanking |
| 1 (VBlank) | 1.09 ms | 4,560 | Vertical blanking |
| 2 (OAM Search) | 19 μs | 80 | Sprite evaluation |
| 3 (Pixel Transfer) | 38-65 μs | 172-289 | Active rendering |

## DMA Timing

| Operation | Duration | Cycles | Notes |
|-----------|----------|--------|-------|
| OAM DMA | 38 μs | 160 | Blocks CPU |

## Instruction Timing

| Category | Cycles | Examples |
|----------|--------|----------|
| Register | 4 | ld a, b, add a, b |
| Immediate | 8 | ld a, $42, add a, $10 |
| Memory Read | 8-12 | ld a, [$C000], ld a, [hl] |
| Memory Write | 8-12 | ld [$C000], a, ld [hl], a |
| Jump | 8-12 | jp $8000, jr .label |
| Call | 18 | call $8000 |
| Return | 16 | ret |
| Push/Pop | 12-16 | push af, pop bc |

## Critical Timing Windows

| Window | Duration | Purpose |
|--------|----------|---------|
| VBlank | 4,560 cycles | VRAM/OAM updates |
| HBlank | 87-204 cycles | VRAM/OAM updates |
| OAM DMA | 160 cycles | Sprite transfer |
