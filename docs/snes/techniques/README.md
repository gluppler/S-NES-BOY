# SNES Techniques

This directory contains applied patterns and reusable approaches for SNES-specific development.

## Purpose

Techniques placed here demonstrate:
- **Hardware-grounded implementation patterns**: Solutions that respect SNES hardware constraints
- **Reusable solutions to common problems**: Patterns that can be adapted for different projects
- **System-adaptive approaches**: Techniques that work within SNES limitations
- **Proven implementation strategies**: Methods verified to work correctly
