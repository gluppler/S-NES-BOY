# 4.1 LR35902 CPU Cheatsheets

## System Applicability

**This cheatsheet applies to:**
- ✅ **Game Boy (DMG): Primary focus
- ✅ **Game Boy Color (CGB): Fully compatible
- ❌ **NES/SNES: Does not apply

## Registers

| Register | Size | Description |
|----------|------|-------------|
| A | 8-bit | Accumulator |
| F | 8-bit | Flags (Z N H C) |
| B, C, D, E, H, L | 8-bit | General-purpose |
| BC, DE, HL | 16-bit | Register pairs |
| SP | 16-bit | Stack pointer |
| PC | 16-bit | Program counter |

## Flags

| Flag | Bit | Meaning |
|------|-----|---------|
| Z | 7 | Zero (result = 0) |
| N | 6 | Subtract (after subtraction) |
| H | 5 | Half-carry (4-bit carry) |
| C | 4 | Carry (8-bit carry/borrow) |

## Instruction Timing (Cycles)

| Instruction | Cycles | Notes |
|-------------|--------|-------|
| ld a, b | 4 | Register to register |
| ld a, $42 | 8 | Immediate load |
| ld a, [$C000] | 12 | WRAM read |
| ld a, [$FF80] | 8 | HRAM read |
| ld [hl], a | 8 | Indirect write |
| ld a, [hl+] | 8 | Indirect read, increment |
| add a, b | 4 | Register add |
| add a, $10 | 8 | Immediate add |
| jp $8000 | 8 | Absolute jump |
| jr .label | 8/12 | Relative jump (not taken/taken) |
| call $8000 | 18 | Subroutine call |
| ret | 16 | Return |
| push af | 16 | Push register |
| pop bc | 12 | Pop register |

## Addressing Modes

| Mode | Example | Cycles |
|------|---------|--------|
| Immediate | ld a, $42 | 8 |
| Register | ld a, b | 4 |
| Absolute | ld a, [$C000] | 12 |
| Indirect | ld a, [hl] | 8 |
| Indirect+ | ld a, [hl+] | 8 |
| Indirect- | ld a, [hl-] | 8 |
| Relative | jr .label | 8/12 |

## Interrupt Vectors

| Interrupt | Vector | Priority |
|-----------|--------|----------|
| VBlank | $0040 | Highest |
| LCD STAT | $0048 | High |
| Timer | $0050 | Medium |
| Serial | $0058 | Low |
| Joypad | $0060 | Lowest |

## Memory Map

| Address | Size | Description |
|---------|------|-------------|
| $0000-$3FFF | 16 KB | ROM Bank 0 |
| $4000-$7FFF | 16 KB | ROM Bank N |
| $8000-$9FFF | 8 KB | VRAM |
| $A000-$BFFF | 8 KB | External RAM |
| $C000-$CFFF | 4 KB | WRAM Bank 0 |
| $D000-$DFFF | 4 KB | WRAM Bank 1 (CGB) |
| $E000-$FDFF | 8 KB | WRAM Mirror |
| $FE00-$FE9F | 160 B | OAM |
| $FF00-$FF7F | 128 B | I/O Registers |
| $FF80-$FFFE | 127 B | HRAM |
| $FFFF | 1 B | IE Register |
