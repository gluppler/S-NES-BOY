# SNES Debug Directory

This directory contains debug utilities and tools for SNES development.

## Purpose

Debug utilities placed here should:
- **Aid development**: Help visualize game state and debug issues
- **Provide runtime information**: Display registers, memory, or game state
- **Support emulator debugging**: Work with emulator debuggers and breakpoints

## Files

- `debug_overlay.asm` – Debug overlay rendering code
- `breakpoints.asm` – Breakpoint definitions for emulator debugging

## Current Status

This directory is currently empty. Debug utilities will be added as needed for development.
